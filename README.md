## Setup
- Clone the Repo
```
git clone https://github.com/Mr-Sunglasses/book.kanishkk.xyz

cd book.kanishkk.xyz
```
- Install mdbook into your system ( using script )
```
cd scripts/

chmod +x install_mdbook.sh

./install_mdbook.sh
```

- Install mdbook into your system ( using make )
```
make install
```

- Run the book
```
make serve
```

- Build for the production
```
make build
```

- Clean the build
```
make clean
```
