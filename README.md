# book.kanishkk.me
An online fingerprint of my knowledge base

## Setup
- Clone the Repo
```
git clone https://github.com/Mr-Sunglasses/book.kanishkk.me

cd book.kanishkk.me
```
- Install mdbook into your system
```
cd scripts/

chmod +x install_mdbook.sh

./install_mdbook.sh
```

- Run the book
```
make serve
```
